Bird Atlas Dashboard

A simple tool for map visualization of Bird Atlases. The map will show the current status of bird count coverage across the chosen geographical area, provided that its feeder Google Sheets are properly filled in.

The data is read from a set of Google Sheets maintained by Birders.



This tool was Originally developed by Vijay Konnackal and now maintained by Bird Count India

For more information or reporting bugs write to skimmer@birdcount.in
